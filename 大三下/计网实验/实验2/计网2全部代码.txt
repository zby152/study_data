//启用RIP协议
	"""
	主要使用的指令如下：
	router rip： 启用RIP协议
	no auto-summary： 关闭子网汇总
	network [接口网段]： 指定接收发送路由表的接口网段范围
	【保证开始时进入 RouterA（config）#  状态再输入以下指令】
	"""
router rip
no auto-summary
version 2
network 202.114.64.0 255.255.255.0
network 202.114.65.8 255.255.255.252
network 202.114.65.0 255.255.255.252
exit

show ip rip 
截图


//启用RIPv2身份验证
	采用md5的方式对RIP协议进行身份验证，主要使用的指令如下：
	key chain name：启用钥匙链
	key id：定义钥匙id
	key-string [密码]：定义密码
	ip rip authentication mode md5：启用md5加密
	【保证开始时进入 RouterA（config）#  状态再输入以下指令】
interface gig 0/1
ip rip authentication mode md5
key chain authRIP
key 1
key-string 1234
int gig 0/1
$ication key-chain authRIP【第一个符号不确定，看终端是否显示】
【【结束这部分注意截图】】
