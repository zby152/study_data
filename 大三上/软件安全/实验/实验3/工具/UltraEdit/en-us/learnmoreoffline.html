What is disconnected/offline activation?

Disconnected activation is a way for you to activate this product without having it directly connect to the internet. If your computer or network uses a firewall that blocks the program or if you do not have an internet connection for this computer, disconnected activation is a way for you activate your title simply and easily.

Is there anything I can try first?

Yes! Some other possible options are adding an exception to your person firewall, temporarily enabling internet, checking if your internet connection is working, or contacting your network administrator for help with connections. However, barring these possibilities, the program must still be activated and offline/disconnected activation is the easiest way.

How does disconnected/offline activation work?

To complete activation without a direct connection, you will generate what is called an "Activation Request" file. Then, you will upload this file to a target web site. If your program is blocked by a personal or network firewall, but your normal web browser works fine, then you may be able to upload the file right on the computer you are currently using. If your computer doesn’t have an internet connection at all, you will have to move the "Activation Request" file to a computer that does have an internet connection.

Once you upload your "Activation Request" file to the server, the server will respond with an "Activation Response" file. You can then save this file and "install" the file via the UI.

Once the "Activation Response" file is installed, your application will be usable!
